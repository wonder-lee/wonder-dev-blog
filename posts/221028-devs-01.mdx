## 내용

### Service 목적

- youtube > 특정 검색어 검색
- 영상 결과 값 중 아래 조건을 필터
  - 50000 > 영상 channel 구독자 수
  - 영상 조회수 > 영상 channel 구독자 수

### Youtube Open Api 이용

- service 목적을 달성하기 위해 필요한 값은 아래와 같습니다.
  - 사용자가 입력한 검색어 → 사용자
  - 검색된 영상들 → youtube
  - 검색된 영상들의 channel의 구독자 수 → youtube
  - 검색된 영상의 조회 수 → youtube
- 특정 검색어 검색 API
  - [https://developers.google.com/youtube/v3/docs/search/list?hl=ko](https://developers.google.com/youtube/v3/docs/search/list?hl=ko)
  - 해당 API에서 영상 channel 구독자수와 영상 조회수를 판별할 수 있는 ID를 수집합니다.
  - channelId, videoId
- 구독자 수 조회 API
  - [https://developers.google.com/youtube/v3/docs/channels/list?hl=ko](https://developers.google.com/youtube/v3/docs/channels/list?hl=ko)
  - 해당 API에서 위 API에서 수집한 channeId를 통해 구독자 수를 조회합니다.
- 영상 조회수 조회 API
  - [https://developers.google.com/youtube/v3/docs/videos/list?hl=ko](https://developers.google.com/youtube/v3/docs/videos/list?hl=ko)
  - 해당 API에서 위 API에서 수집한 videoId를 통해 조회수를 조회합니다.
